How it works:

  • Conic-Gradient: Creates the color sweep around the center.
  • @property: Allows the browser to animate the angle variable smoothly.
  • Mask-Composite: Removes the center of the element so the background doesn't cover the content.
  • Drop-Shadow: Adds the actual neon light "spill".
